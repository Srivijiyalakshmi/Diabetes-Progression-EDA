# Diabetes Progression EDA
## Description
This repository contains an **Exploratory Data Analysis (EDA)** of a diabetes dataset. The analysis explores relationships between patient physiological attributes and diabetes progression, uncovering key factors that influence disease outcomes.

## Project Overview
- **Objective:** Analyze diabetes dataset to identify features most associated with disease progression.
- **Dataset:** Diabetes dataset containing patient age, sex, BMI, blood pressure, serum measurements (s1–s6), and disease progression.
- **Tools Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Features Analyzed
**Feature: Description**
- **age:** Age of the patient
- **sex:** Gender of the patient
- **bmi:** Body Mass Index
- **bp:** Blood pressure
- **s1–s6:** Serum measurements
- **progression:** Diabetes disease progression after one year
  
## Key Insights
- BMI, blood serum measurement **s5**, and blood pressure are the strongest predictors of diabetes progression.
- Age and sex have weaker correlations with disease progression.
- Visualizations including histograms, scatter plots, correlation heatmaps, and pairplots were used to reveal patterns and trends.

## Files
- `Diabetes.ipynb` → Jupyter Notebook with full analysis and visualizations
- `Diabetes-Progression-EDA-Report.pdf` → Comprehensive report summarizing methods, insights, and conclusions
